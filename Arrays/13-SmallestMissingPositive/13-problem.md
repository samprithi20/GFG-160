# Smallest Positive Missing Number

---

## Problem Statement

You are given an integer array `arr[]`.  
Your task is to find the **smallest positive number** missing from the array.

> üîπ Note: Positive numbers start from 1.  
> üîπ The array may contain negative integers as well.

---

## Examples

### Example 1:
**Input:**  
`arr[] = [2, -3, 4, 1, 1, 7]`  
**Output:**  
`3`  
**Explanation:**  
The smallest positive missing number is `3`.

---

### Example 2:
**Input:**  
`arr[] = [5, 3, 2, 5, 1]`  
**Output:**  
`4`  
**Explanation:**  
The smallest positive missing number is `4`.

---

### Example 3:
**Input:**  
`arr[] = [-8, 0, -1, -4, -3]`  
**Output:**  
`1`  
**Explanation:**  
There are no positive numbers, so the smallest positive missing number is `1`.
